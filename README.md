### 🔌 Socket.io 핵심 함수 정리

**socket.emit("이름", 데이터) : 상대방에게 신호 보내기**

**socket.on("이름", (data) => { ... }) : 특정 신호 수신 시 코드 실행**

**io.emit("이름", 데이터) : 서버에 접속한 모든 사용자에게 신호 전송**
