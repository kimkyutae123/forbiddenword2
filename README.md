**금칙어 게임 사이트**

**1**. 1번은 2번한테 2번은 3번한테 3번은 4번한테 4번은 1번한테 전해지는 로직
**2**. 무조건 1번이 방장 1번이 게임 시작 버튼을 눌러서 다 들어온 걸 확인 후 시작함
**3**. 소켓.io로 연동은 가장 마지막에 개발하고
**4**. 제작자가 기본값으로 1,2,3,4번에 캐릭터 사진을 넣어 놓고 사용자들이 들어오면 1,2,3,4를고르는 방식 
**5**. 사용자가 금칙어를 말했을 시 사용자는 채팅창을 사용할 수 없고 게임이 끝날때까지 관전상태
(ex.1,2,3,4번이서 게임진행중 3번이 2번이 정해준 금칙어를 말해 게임에 아웃됐을 경우 3번의 금칙어를 채팅창에 공개후 3번은 관전상태로 게임진행 최종1인이 남을 때 까지 가리는 방식)
